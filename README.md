# People Analytics — Mapa de Colaboradores

Dashboard interativo de People Analytics desenvolvido com Streamlit e Plotly.

## Dados

- **Fonte:** `Base_dados_rh.xlsx`
- **164 colaboradores** · 100% ativos · Referência: Março 2026

## Funcionalidades

- Filtros interativos por Departamento, Performance, Satisfação e Canal de Recrutamento
- 6 KPIs dinâmicos atualizados por filtro
- Visualizações: distribuição departamental, gênero, faixa etária, contratações por ano, performance, satisfação, níveis hierárquicos, faixa salarial, top cargos e retenção

## Executar localmente

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy

Hospedado no [Streamlit Community Cloud](https://streamlit.io/cloud).
